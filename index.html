Sistema de Vendas Inteligente

Organize suas vendas, gerencie setores e visualize relatórios de forma segura e rápida

Vantagens do Sistema

Login Seguro

Autenticação via código enviado por e-mail para proteger seus dados e controlar o acesso.

Organização por Setores

Mantenha vendas, relatórios e CSVs organizados por departamento ou setor da empresa.

Histórico de Vendas

Registre e consulte todas as vendas realizadas, com exportação fácil para CSV.

Dashboard Visual

Gráficos claros e interativos para acompanhar vendas por data ou setor.

Baixe o Sistema

Compatível com Python 3.x e bibliotecas: ttkbootstrap, bcrypt, matplotlib

Download do Sistema

Como instalar:

  1. Baixe o arquivo ZIP acima.
  2. Extraia para uma pasta de sua preferência.
  3. Certifique-se de ter Python 3 instalado.
  4. Instale as bibliotecas necessárias: pip install ttkbootstrap bcrypt matplotlib
  5. Abra o terminal na pasta extraída e execute: python sistema_vendas.py
