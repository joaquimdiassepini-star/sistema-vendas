# Sistema de Vendas em Python

Um sistema completo de gerenciamento de vendas para empresas, desenvolvido em **Python** usando **Tkinter** e **ttkbootstrap**. Ele permite controle de produtos, vendas, histórico, dashboards visuais e segurança via login por e-mail.

---

## 🔹 Funcionalidades

- Cadastro de usuários com senha segura (mínimo 6 caracteres).
- Login com envio de código de autenticação via e-mail.
- Diferenciação de usuários por função: **Admin** e **Vendedor**.
- Adição, edição e remoção de vendas com cálculo de desconto automático.
- Histórico de vendas e exportação para CSV.
- Organização de arquivos e relatórios por setor/departamento.
- Dashboard com gráficos de vendas por data ou setor.
- Interface moderna e responsiva com **ttkbootstrap**.
